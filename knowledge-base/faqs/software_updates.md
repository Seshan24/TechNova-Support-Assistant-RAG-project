# Software Updates FAQ

## How often does TechNova release software updates?
- **Major updates:** Once per year (autumn), bringing new features
- **Minor updates:** Every 2–3 months, with smaller features and improvements
- **Security patches:** Monthly, on the second Tuesday of each month
- **Critical security patches:** As needed, often within 48 hours of a vulnerability disclosure

## How long will my Nova device receive updates?
TechNova guarantees software updates for **at least 5 years** from the product's launch date. This includes both feature updates and security patches.

Specifically:
- **Nova Phone X1** (launched 2024-03): Updates guaranteed until at least 2029-03
- **Nova Laptop Pro 14** (launched 2024-08): Updates guaranteed until at least 2029-08
- **Nova Watch Series 3** (launched 2024-05): Updates guaranteed until at least 2029-05
- **Nova Tablet 11** (launched 2024-11): Updates guaranteed until at least 2029-11

## How do I install an update?
**Phone, Tablet, Watch:**
- Settings > System > Software Update > Check for Update

**Laptop:**
- System Preferences > Software Update

Make sure your device is plugged in and connected to Wi-Fi.

## Can I roll back an update?
Generally **no**. Once you install an update, it cannot be reverted. If a new update is causing issues, contact support — we can sometimes provide a workaround or expedite a patch.

## My update is stuck. What do I do?
1. Make sure you have at least 5GB of free storage.
2. Connect to a reliable Wi-Fi network (not cellular).
3. Plug the device into power.
4. If the download is stuck at the same percentage for more than 30 minutes, force restart the device and try again.

## Will an update wipe my data?
No. Major and minor updates preserve all your data. However, **always back up before any update** — corruption is rare but possible.
