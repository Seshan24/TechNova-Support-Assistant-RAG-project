# NovaOS 5.1 Release Notes

**Release date:** July 15, 2024
**Affected device:** Nova Phone X1
**Update size:** 1.4 GB

## What's New

### Camera improvements
- New **Night Portrait** mode using AI-assisted denoising
- Manual focus controls in Pro camera mode
- Improved low-light video stabilization

### Battery & Performance
- Improved background process management — average **8% battery life improvement** on standby
- Faster app launch times (up to 15% faster for some apps)
- Reduced background CPU usage when GPS is active

### Privacy
- New **App Privacy Report** showing which apps accessed sensitive data and when
- More granular location permission options (precise vs approximate)
- New indicator dot when microphone or camera is in use

### Bug Fixes
- Fixed an issue where Bluetooth multipoint could disconnect after lock screen
- Fixed an issue where Wi-Fi 6E networks weren't always preferred over slower bands
- Fixed an issue where some images in the gallery appeared as black thumbnails
- Fixed a rare crash when using the calculator with very large numbers

## Known Issues
- Some third-party launchers may require an update to support the new App Privacy Report
- Spotify users may need to re-grant Bluetooth permissions on first launch

## Security
- 12 CVEs patched (see security.technova.example.com/advisories for details)

## How to Install
Settings > System > Software Update > Check for Update
